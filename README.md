El presente proyecto, titulado “Plataforma de Registro y Control de Eventos”,
se desarrolla como requisito académico para la asignatura de Ingeniería de Requisitos, del programa de Ingeniería de Software del Campus Pasto. 
Su propósito es aplicar los principios y metodologías del análisis y gestión de requerimientos de software,
mediante la definición, documentación y validación de los procesos necesarios para el desarrollo de una solución tecnológica
que optimice el registro, la organización y el seguimiento de eventos institucionales o corporativos.
Este proyecto busca fortalecer las competencias del estudiante en la identificación de necesidades del usuario, 
el diseño de casos de uso y la elaboración de especificaciones funcionales y no funcionales dentro del ciclo de vida del desarrollo de software.
